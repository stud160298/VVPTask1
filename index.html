Команды Git
Команда Описание
git init - creates a new Git repository
git clone url - clones a repository into a newly created directory
git add file - add file to repository
git add . - adds new or changed files in your working directory
git commit -m "comment" - takes a snapshot of the current state of the changes added to the indexed files section for the project
git commit -a -m "comment" - automatically takes a snapshot of the current state of changes for the project
git push url - the "remote" repository that is destination of a push operation
git status - displays the state of the working directory and the staging area
git log - utility tool to review and read a history of everything that happens to a repository
git branch branch_name - create a branch for every task or issue you work on
git checkout branch_name - directly checkout a branch
git checkout -b branch_name - create and directly checkout a branch
git remote - create, view, and delete connections to other repositories
